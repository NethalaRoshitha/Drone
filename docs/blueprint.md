# **App Name**: AgriSmart AI

## Core Features:

- Home Navigation & Feature Selection: A user-friendly home page with clear buttons to navigate between 'Crop Recommendation' and 'Plant Disease Detection' features.
- Crop Recommendation Data Entry: An interactive form allowing users to input specific environmental data (Nitrogen, Phosphorus, Potassium, Temperature, Humidity, pH, Rainfall) for crop optimization.
- AI-powered Crop Prediction Tool: A backend service that utilizes an AI tool to process input environmental data and provide intelligent recommendations for optimal crop types, suitable fertilizers, and actionable cultivation tips.
- Plant Disease Image Uploader: A dedicated interface for users to easily upload plant images for detailed disease analysis.
- AI-powered Disease Diagnosis Tool: A backend service that employs an AI image recognition tool to identify plant diseases from uploaded images, predict disease confidence levels, and suggest effective treatment or cure methods.
- Results Display & Feedback: A clean and intuitive display area to present the results of crop recommendations and disease diagnoses, including specific tips and actionable advice.

## Style Guidelines:

- The primary color, reflecting growth and nature with a modern twist, is a muted forest green (#3F963F). Its hue provides a calming yet authoritative feel.
- A subtle, almost off-white background color (#EEF4EE) with a faint green tint maintains a clean aesthetic while harmonizing with the primary palette.
- An energetic accent color, a vibrant mint green (#4DD982), is used for calls to action and highlights, bringing an element of fresh discovery and optimism.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and highly readable design suitable for both data presentation and navigational text.
- A centered, responsive layout ensures a consistent and professional user experience across all device sizes, framed by a subtle, modern gradient background.
- Professional and crisp outline icons should be used to represent actions and categories, reinforcing the app's clean and functional design.
- Subtle and smooth transition animations will be implemented for page navigation and when results are loading, enhancing the user interaction without distraction.
